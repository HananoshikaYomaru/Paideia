## Development

Run the [Elysia](https://elysiajs.com) + [Vite](https://vitejs.dev/) dev server:

```sh
bun dev
```

## Production

First, build your app for production:

```sh
bun run build
```

Then run the [Elysia](https://elysiajs.com) app in production mode (with `NODE_ENV=production`):

```sh
bun start
```
